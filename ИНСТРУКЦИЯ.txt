============================================
   ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА ДЛЯ WILDBERRIES FBS
============================================

1. УСТАНОВИТЕ ЗАВИСИМОСТИ
   Откройте терминал (PowerShell или cmd) и выполните:

   cd C:\Users\User\Desktop\wildberries_fbs_bot
   pip install -r requirements.txt

2. НАСТРОЙТЕ .env ФАЙЛ
   Откройте файл .env (лежит в папке wildberries_fbs_bot)
   и укажите ваш токен от @BotFather:

   BOT_TOKEN=ваш_токен_сюда

   Остальные параметры можно оставить по умолчанию:
   - DESTINATION_OFFICE_ID=0   — ID офиса приёмки (узнать в личном кабинете WB)
   - POLL_INTERVAL=30           — интервал проверки новых заказов (сек)
   - DATA_DIR=data              — папка для хранения данных

3. ЗАПУСТИТЕ БОТА
   В терминале выполните:

   cd C:\Users\User\Desktop\wildberries_fbs_bot
   python bot.py

4. ИСПОЛЬЗОВАНИЕ В ТЕЛЕГРАМ
   - Найдите бота в Telegram по его username (который вы задали через @BotFather)
   - Отправьте команду /start
   - Введите ваш API-ключ Wildberries (токен категории "Marketplace")
   - Бот начнёт автоматически проверять новые заказы каждые 30 секунд

5. ДОСТУПНЫЕ КОМАНДЫ
   /start    — Начать работу (ввести API-ключ)
   /help     — Показать справку
   /set_key  — Сменить API-ключ
   /check    — Проверить заказы вручную
   /status   — Статус подключения к API

6. ОСТАНОВКА БОТА
   Нажмите Ctrl+C в терминале, где запущен бот.
